## venv installation
```shell
$ python -m venv --system-site-packages .\venv
```
to start venv type:   
```shell
$ &"./venv/Scripts/Activate.ps1"
```
$ to get out of venv type:
```shell
$ deactivate
```
if error in stating, type in admin cmd:
```shell
$ Set-ExecutionPolicy Unrestricted -Scope Process
```
